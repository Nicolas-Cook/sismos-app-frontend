## Configuración inicial

### Prerrequisitos

- Node.js

### Instalación

#### React

1.  Navega a la carpeta de la aplicación React:  
    `cd …/sismos-app-frontend`
2.  Instala las dependencias necesarias:  
    `npm install`
3.  Inicia el servidor de desarrollo de React:  
    `npm start`
